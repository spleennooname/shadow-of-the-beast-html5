## Panda.js HTML5 game engine

Free HTML5 game engine for mobile and desktop with Canvas and WebGL rendering.

http://www.pandajs.net

### Install

    $ sudo npm install -g pandatool
    $ mkdir MyPandaGame
    $ cd MyPandaGame
    $ panda install
